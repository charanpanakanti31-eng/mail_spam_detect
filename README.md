# Spam Mail Detection using Machine Learning

## 📌 Project Overview
This project detects whether a message is Spam or Not Spam using Machine Learning.

## ⚙️ Technologies Used
- Python
- Scikit-learn
- Pandas
- Streamlit

## 📊 Model Details
- Algorithm: Multinomial Naive Bayes
- Accuracy: 96.6%

## 🚀 How it Works
1. Input a message
2. Model processes text using TF-IDF
3. Predicts Spam or Not Spam

## 📸 Output
Example:
- "Congratulations! You won a lottery" → Spam
- "Are you coming to class?" → Not Spam
